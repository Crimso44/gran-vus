ОБНОВЛЕНИЕ КОМПЛЕКСА ПО "ГРАН-БРОНИРОВАНИЕ" ОТ ВЕРСИИ 3.5 ДО ВЕРСИИ 3.73.

ИСТОРИЯ ВЕРСИЙ:

ВЕРСИЯ 3.73:
  - В отчёте "Форма № 6 организации" в поле "за_____год" помещается либо
    предыдущий год относительно года, содержащегося в поле "Дата представления
    отчета" карточки организации, либо текщий год в зависимости от флага
    "\\Администрирование\Установки\Общие\Подотчетные организации представляют
    отчет после Нового Года".
  - В отчёте "Общие списки организаций" добавлены 2 поля: "Всего работающих" и
    ИНН.
  - В отчёте "Форма № 6 организации" в поля "должность" и "ФИО" подписавшего
    документ ставится должность и ФИО руководителя организации из карточки
    организации.
  - В отчёте "Форма № 6 организации" слово "Секретно" заменено на "Гриф по
    заполнении".
  - В отчёте "Форма № 6 организации" в числовой части таблицы размер шрифта
    сделан 8 вместо 6.
  - В отчёте "Форма № 6 ВУО" в поле "за_____год" помещается либо
    предыдущий год относительно года, содержащегося в поле "Дата представления
    отчета" карточки организации, либо текщий год в зависимости от флага
    "\\Администрирование\Установки\Общие\Подотчетные ВУО представляют
    отчет после Нового Года".
  - В отчёте "Форма № 6 ВУО" в поля "должность" и "ФИО" подписавшего
    документ ставится должность и ФИО руководителя ВУО из карточки
    ВУО.
  - В отчёте "Форма № 6 ВУО" в числовой части таблицы размер шрифта
    сделан 8 вместо 6.
  - Исправлена "Ошибка при обращении к базе данных", возникавшая при длинных
    значенияхКогда в поле карточки организации "№ в ВУО".
  - Исправлена процедура "\\Качествееные характеристики ВУО\Проверка данных о
    распределении персонала ..." для строки 2 "Специалистов всего".
  - В карточке организации по кнопке "Расчетная численность по категориям"
    данные для небронирующих организаций берутся из строки 26, а не строки 25.
  - Добавлен справочник "Военкоматы".
  - В карточке организации добавлено поле "ОВК".
  - В карточке организации поля "Признак.." увеличены до 100 символов.
  - В справочнике мест регистраций "МНС" заменено на "ФНС".
  - В фильтр по организациям редактор значения показывает старое значение.
  - Отчёт по формам собственности объединяет Федеральную собственность и
    собственность субъектов федерации под общим показателем "Государственная
    собственность".
  - Отчёт "Численность по видам экономической деятельности" считает процент 
    забронированных от количества граждан запаса.
ВЕРСИЯ 3.72:
  - Внесены исправления в отчеты "Список организаций, ведущих бронирование".
ВЕРСИЯ 3.71:
  - Список организаций теперь можно фильтровать по разделу ОКВЭД.
  - В мастере выборки теперь для строковых значений можно задавать условия
    "НАЧИНАЕТСЯ С" и "ЗАКАНЧИВАЕТСЯ НА".
  - Появилясь возможность открывать выборку в новом окне.
  - Изменился формат выгрузки данных в Excel.
  - Расчётные данные "по категориям персонала и отношению к воинской повинности"
    переименованы в "по категориям персонала и отношению к воинской
    обязанности".
  - Изменён способ расчёта отчёта "Сводные показатели ВУО".
  - При расчёте общих колличественных характеристик ВУО теперь не учитываются
    организации, снятые с учёта.
  - При заполнении формы №6 строка №2 теперь не пересчитывается автоматически
    при изменении строк 3-10.
  - При проверке формы №6 теперь рассматривается значение строк 21-23 по
    отношению к строке 24.
  - Исправлен недочёт в подсчёте строки "Прочее" при формировании сведений о
    численности по видам экономической деятельности.
  - В карточке ВУО закладка "Органы субъекта" переименована в "Органы
    управления".
  - В карточке ВУО на закладке "Органы субъекта" в таблице значений пункт
    "Орган исполнительной власти" переименован в "Руководящий орган".
  - Прочее.

ВЕРСИЯ 3.6.1:
  - Исправлена ошибка при редактировании даты регистрации организации на рас-
    ширенном диалоговом окне.
  - Исправлена ошибка при сохранении сведений об организации, возникающая при
    наличии дублирующейся информации в сведениях об основаниях и перечнях.
  - Исправлена ошибка при сохранении контактной информации. Теперь для сохране-
    ния записи о руководителе или ВУР достаточно заполнить любое значение, оп-
    ределяющее сведения о персоне (фамилия, имя, отчество, телефон или должность).

ВЕРСИЯ 3.6:
  - Полностью переработана карточка организации. Система закладок заменена на
    дополнительные диалоги с более полными сведениями.
  - Добавлена возможность фильтрации списка организации по диапазонам значений,
    для полей содержащих суффиксы (№ ОКБВ).
  - Раширены длины полей "ИНН" и "Основной ОКВЭД (ОКОНХ)" до 15 символов.
  - Добавлены поля ОКАТО для фактического и почтового адреса.
  - В сведениях о контактной информации поле "ФИО" разделено на фамилию, имя и
    отчество.
  - Обновлено поле просчета форм-6 для записей созданных до версии 3.5 на значение
    классификатора.
  - Обновлены хранимые процедуры MSSQL импорта организации и ВУО.
